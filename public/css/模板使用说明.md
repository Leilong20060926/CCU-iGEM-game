# CSS 模板使用指南

## 两个模板文件

### 1. **text-box.css** - 文字框模板
适用于简单的故事文字框。如：`1-1.html`、`2-1.html`、`3-1.html` 等。

**特点：**
- 简单的方形文字框
- 白色三角形箭头
- 无角色名字标签

**使用示例（在 .html 文件中）：**

```html
<head>
  <link rel="stylesheet" href="../../public/css/shared.css" />
  <link rel="stylesheet" href="../../public/css/text-box.css" />
  <style>
    /* 可在这里覆盖颜色 */
    .story-text-box {
      background: rgba(60, 100, 60, 0.85);  /* 改为你想要的背景颜色 */
      color: white;                          /* 改为你想要的文字颜色 */
    }
    
    .story-text-arrow {
      border-top-color: white;              /* 改为你想要的箭头颜色 */
    }
  </style>
</head>
```

---

### 2. **dialogue-box.css** - 对话框模板
适用于带角色对话的对话框。如：`1-2-c.html`、`1-2.html` 等。

**特点：**
- 圆形的对话框，带气泡尾巴
- 有角色名字标签
- 深色文字（适合浅色背景）

**使用示例（在 .html 文件中）：**

```html
<head>
  <link rel="stylesheet" href="../../public/css/shared.css" />
  <link rel="stylesheet" href="../../public/css/dialogue-box.css" />
  <style>
    /* 可在这里覆盖颜色 */
    .story-text-box {
      background: rgba(180, 220, 180, 0.95);  /* 改为你想要的背景颜色 */
      color: #333;                             /* 改为你想要的文字颜色 */
    }
    
    .story-text-box::before {
      border-bottom-color: rgba(180, 220, 180, 0.95); /* 尾巴颜色要与背景相同 */
    }
    
    .character-name {
      background: rgba(100, 180, 100, 0.9);  /* 改为你想要的标签颜色 */
    }
    
    .story-text-arrow {
      border-top-color: #333;                 /* 改为你想要的箭头颜色 */
    }
  </style>
</head>
```

**HTML 结构（必须）：**

```html
<div class="story-text-box" id="storyTextBox">
  <div class="character-name">角色名字</div>
  <div id="storyText" class="story-text">对话文字</div>
  <a href="#" class="story-text-arrow"></a>
</div>
```

## 颜色参数参考

### 各角色推荐颜色

| 角色 | 背景颜色 | 文字颜色 | 标签颜色 | 箭头颜色 |
|------|--------|--------|--------|--------|
| 蛾（瘤野螟） | rgba(180, 220, 180, 0.95) | #333 | rgba(100, 180, 100, 0.9) | #333 |
| 水稻 | rgba(200, 180, 140, 0.9) | #333 | rgba(150, 120, 60, 0.85) | #333 |
| 农民 | rgba(160, 140, 110, 0.9) | white | rgba(100, 80, 50, 0.85) | white |

---

## 重要提示

1. **对话框模板**中，`::before` 的 `border-bottom-color` 必须与背景颜色相同，否则气泡尾巴会显示不对。

2. 使用模板时，只需在 HTML 的 `<style>` 中写入你要改的颜色部分，其他的默认样式会从模板中继承。

3. 如果要调整框的大小、位置等，可以在自己的 `<style>` 中覆盖这些属性。
